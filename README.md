# Python Port Scanner

A beginner-friendly cybersecurity project that demonstrates how a simple TCP port scanner can be progressively improved using Python.

## Versions

|  Version |        Feature         |
|----------|------------------------|
|    V1    | Basic TCP Port Scanner |
|    V2    |   Port Range Support   |
|    V3    | Multithreaded Scanning |
|    V4    |    Banner Grabbing     |
|    V5    |   JSON Report Export   |

## Blog Series

- Version 1: Basic Port Scanner
- Version 2: Port Range Support
- Version 3: Multithreading
- Version 4: Banner Grabbing
- Version 5: JSON Reporting
